# Panel Admin Facil

Version simplificada del panel para que el cliente vea todo mas claro y con menos tecnicismos.

## Arranque

```powershell
node backend/admin_panel_facil/server.js
```

Abre:

```text
http://localhost:5052
```

## Que cambia en esta version

- Lenguaje mas simple.
- Menos ruido visual.
- Formularios mas directos.
- Edicion de listas con una linea por elemento.
- Botones grandes para gestionar reservas.

## Importante

Esta version tambien usa `database.sqlite` y guarda datos reales, pero no toca ningun archivo existente de la web principal.
